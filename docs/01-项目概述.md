# 项目概述

### 1.1 业务背景

退货量较大时，客服需要一条一条手工将快递单号录入表格，耗时且易出错。希望通过拍照识别快递单信息，自动提取关键字段并写入飞书多维表格，减少手工录入工作量。

### 1.2 建设目标

| 目标 | 说明 |
|------|------|
| 拍照即录入 | 拍摄快递单照片，系统自动 OCR 提取信息 |
| 自动写入飞书 | 识别结果自动写入飞书多维表格，无需手工敲入 |
| 批量处理 | 支持一次上传多张照片，批量识别录入 |
| 人工校验 | 识别结果可人工确认/修改后再写入，保证准确性 |

### 1.3 核心流程

```
[拍摄快递单照片] ──上传──▶ [双引擎OCR识别] ──提取──▶ [交叉验证]
         │                                              │
         │    ┌─────────────────────────┐              │
         │    │ 智谱OCR(引擎A) + 阿里云OCR(引擎B) │            │
         │    └─────────────────────────┘              │
         │                                              │
         ▼                                              ▼
[飞书多维表格] ◀──写入── [人工确认/修改] ◀──结构化信息──
```
