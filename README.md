# 退运智录（ReturnVision）- 退货OCR解决方案

> 版本：v2.0 | 日期：2026-07-10 | 状态：方案设计（双引擎交叉验证架构，智谱OCR+阿里云OCR均已联网验证）
> 上线地址：https://returnvision.jxing.tech

## v2.0 更新内容

- **OCR引擎升级**：从单引擎（阿里云）改为双引擎交叉验证（智谱OCR引擎A + 阿里云引擎B）
- **修正GLM-5.2错误**：GLM-5.2为纯文本模型不支持图片，已更正为智谱OCR解析工具
- **新增DeepSeek LLM分析**：语义校验 + 退货原因提取 + 智能分类，~0.0004元/次
- **新增腾讯云COS**：图片持久化存储，URL供双引擎OCR和飞书引用
- **数据库改为MySQL**：生产级可靠性，替代SQLite
- **新增飞书机器人通知**：写入飞书后自动发卡片消息到群
- **新增交叉验证引擎**：双引擎并行识别 + 字段比对，一致时准确率接近99.9%
- **后端语言改为Java**：Python/FastAPI 改为 Java 21/Spring Boot 3.x + MyBatis-Plus
- **新增五层保障体系**：交叉验证 -> 置信度判断 -> 格式校验 -> DeepSeek语义校验 -> 人工确认
- **成本优化**：智谱0.01元/次（原阿里云0.0825元/次），月成本降低约80%
